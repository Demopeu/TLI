# 표현식과 값

## 표현식

> 값, 변수, 연산자 등을 조합하여 계산되고 결과를 내는 코드 구조

> 표현식이 **평가**되어 값이 반환

**평가**

```

- 표현식이나 문장을 실행하여 그 결과를 계산하고 값을 결정하는 과정

- 표현식이나 문장을 순차적으로 평가하여 프로그램의 동작을 결정

```

**문장**

```

- **실행 가능**한 독작을 기술하는 코드

>> 조건문, 반복문, 함수 정의 등

```

## 타입

> 값이 어떤 종류의 데이터인지, 어떻게 해석되고 처리되어야 하는지를 정의

> **값**과 **값에 적용할 수 있는 연산**

### 데이터 타입

#### Numeric Type

- int

- float

- complex

#### Sequence Type

- list

- tuple

- range

#### Text Sequence Type

- str

#### Set Type

-  set

### 산술연산자

- ``//`` 몫

- ``%`` 나머지

- ``**`` 지수

#### 우선순위

```
** > - > */,//,% > +,-

```
