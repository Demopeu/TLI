# 표현식과 값

## 표현식

> 값, 변수, 연산자 등을 조합하여 계산되고 결과를 내는 코드 구조

> 표현식이 **평가**되어 값이 반환

**평가**

```

- 표현식이나 문장을 실행하여 그 결과를 계산하고 값을 결정하는 과정

- 표현식이나 문장을 순차적으로 평가하여 프로그램의 동작을 결정

```

**문장**

```

- "실행가능"한 동작을 기술하는 코드

>> 조건문, 반복문, 함수 정의 등

```

> 할당 연산자(=)

```

```

#### 변수의 재할당

```

이미 할당이 끝났기 때문에 number가 바뀌어도 할당은 바뀌지 않음
```

# Data Types

`` 분류는 위쪽에서 확인 ``

## 데이터 타입이 필요한 이유

> 값들을 구분

> 변수의 의도를 더 쉽게 이해 할 수 있고, 오류 예방 가능


## Numeric Types

### int

> 정수를 표현하는 자료형

#### 진수 표현

- 2진수 : 0b

- 8진수 : 0o

- 16진수 : 0x

### float

> 실수를 표현하는 자료형 -> 실수에 대한 *근삿값*

``유한 정밀도 - 무한대의 숫자를 용량 한정으로 저장 불가능 ``

### 실수 연산 시 주의사항

> 컴퓨터는 2진수 사용, 사람은 10진법 사용

> 근삿값이기 때문에 에러 혹은 오류 생길 수 있음
``Floating point rounding error ``

### 실수 연산 시 해결책

> 두 수의 차이가 매우 작은 수보다 작은지를 확인

> math 모듈 활용
``isclose()``

> 지수 표현 시 e or E를 사용

## Sequence Types 특징

- 인덱싱

- 슬라이싱

- 길이

- 반복

### str

> 순서가 있는 변경 X한 자료형

#### Esacpe sequence

- ``\n `` 줄 바꿈

- ``\t `` 탭

- ``\\ `` 백슬래시

- ``\' `` 작은 따옴표, ``\" `` 큰 따옴표

#### String lnterpolation

```
name = '김동현'
age = 19
area = '부산'
print(f'안녕하세요{name} {age} {area}')
```

> 인덱싱이 있기에 슬라이싱, 길이 ㄱㄴ

**인덱스(index)**

```
시퀀스 내의 값들에 대한 고유 번호
앞:0부터 +
뒤:-1부터 - 
```

**슬라이싱(slicing)**

```
일부분 선택하여추출
ex
my_str = 'hello'
my_str[2:4]
my_str[:3]
my_str[3:]
my_str[0:5:2]
my_str[::-1] - 'olleh'
```

`` 문자열은 중간 변경 불가 ``
