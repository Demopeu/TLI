# 이진 탐색 트리

> 탐색작업을 효육적으로 하기 위한 자료구조

> 모든 원소는 서로 다른 유일한 키

> 중위 순회 하면 오름차순으로 정렬된 값을 얻을 수 있음

## 삽입 연산

- 먼저 탐색 연산 수행

    > 삽입할 원소와 같은 원소가 트리에 있으면 삽입 불가능

    > 탐색 실패하면 그 위치에 삽입

- 탐색, 삽입, 삭제 시간은 트리의 높이 만큼 소요

## 검색 알고리즘

1. 순차 검색 : O(N)

2. 정렬된 배열 순차 검색 : O(N)

3. 이진 탐색 : O(logN)

4. 이진 탐색 트리 평균: O(logN)

5. 해쉬 검색 : O(1)

    > 추가 저장 공간 필요
